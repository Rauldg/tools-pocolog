== What is pocosim-log ?
This package allows to manipulate (read and write) the log files generated by
the pocosim[1] simulator. These log files contain streams of timestamped
binary data, along with a Typelib[2] XML definition of the binary data (thus
ensuring that the file will remain readable). A tool, pocosim-logger, allows
to generate this kind of file from posters of Genom[3] modules

[1] http://www.laas.fr/~sjoyeux/simulation.php

[2] http://www.laas.fr/~sjoyeux/doc/api/typelib and
    http://www.laas.fr/~sjoyeux/doc/api/typelib-ruby

[3] http://softs.laas.fr/openrobots
